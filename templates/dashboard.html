How to Install MindPulse Extension

1

Download the Extension

Click the Download Extension button above. A MindPulse_Extension.zip file will be saved to your Downloads folder.

2

Extract the ZIP File

Right-click the downloaded ZIP → Extract All. Remember the extracted folder location.

3

Open Chrome Extensions Page

Open a new tab in Chrome and go to chrome://extensions/

4

Enable Developer Mode

Toggle Developer mode ON using the switch in the top-right corner of the extensions page.

5

Load the Extension

Click Load unpacked → Select the extracted folder containing manifest.json. The extension will appear in your extensions list.

6

Pin & Configure

Click the puzzle icon in Chrome toolbar → Pin MindPulse. Open it, paste your API Token and Backend URL, then click Sync Now!

Tip: Generate a token using the Generate Token button above, then use it in the extension popup along with your Backend URL: {{ request.host_url }}
